SGI

O projeto SGI foi o último projeto feito como forma de estudos, e também o projeto com mais tecologias envolvidas até o presente momento, esse projeto consiste no desenvolvimento de um sistema de gerenciamento de clientes. Isso é possivel após o usuário acessar um painel utilizando um login (e-mail) e uma senha, ao efetuar o login ele é redirecionado a uma base de dados onde consegue visualizar todos os clientes cadastrados no sistema bem como seu status, tem a possibilidade de ver os clientes de forma individual, fazer alteração nos dados dos clientes cadastrados, realizar o cadastro de novos clientes e assim armazená-los diretamente no banco de dados, tudo isso passando por validações para não serem inseridos dados inválidos, validações essas feitas no lado clienete e no lado servidor. Este sistema segue em evolução e ganhara com o tempo ainda mais funcionalidades
Tecnologia utilizadas: HTML, CSS, Bootstrap, JavaScript, PHP, SQL, MySql.

Sobre mim

Sou Desenvolvedor Full Stack, atualmente desenvolvo alguns projetos como forma de aprendizado, crio soluções sempre prezando pelas boas práticas de programação.

Atualmente eu utilizo a linguagem de marcação HTML e as folhas de estilo do CSS, também utilizei em alguns projetos o framework Bootstrap para auxiliar no desenvolvimento de aplicações responsivas. Em relação a linguagens de programação, tenho utilizado JavaScript e PHP, tenho também algumas automatizações feitas utilizando Python.

Meus projetos são feitos com essas linguagens sem a utilização de frameworks, pois acredito que aprendendo a linguagem de forma nativa se torna mais fácil e eficaz a resolução de problemas que possam acontecer durante o desenvolvimento e testes.
